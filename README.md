Typing Speed Test


A simple Python application for testing your typing speed. This application presents you with a random paragraph, and you need to type it as quickly and accurately as possible within a specified time limit. After the time is up, the application calculates and displays your typing accuracy and words per minute (WPM).

View Demo

Features
Randomly generated paragraphs for typing practice.
Adjustable timer (30 seconds or 60 seconds).
Real-time countdown timer.
Calculation of typing accuracy and words per minute (WPM).
User-friendly GUI using Tkinter.
Requirements
Python 3.x
Tkinter (usually included with Python)

Usage
Launch the application using the installation instructions above.

Choose your preferred time limit (30 seconds or 60 seconds) by selecting the corresponding radio button.

Click the "Start Timer" button to begin the typing test.

You will be presented with a random paragraph in the text area. Type the paragraph as quickly and accurately as possible.

The countdown timer will display the time left.

After the time is up, the application will display your typing accuracy and words per minute (WPM).

You can restart the test by clicking the "Start Timer" button again.
